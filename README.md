# HashCalculator [v0.5.0-alpha]  
  
Hash計算機でHashを計算できます。  
また、ファイルを右クリックしてHash for ContextMenu(F)をクリックし、  
Hashを選択または*をクリックするとそのファイルのHashを計算できます。  
  
対応Hashアルゴリズム  
・MD5  
  
・SHA1  
・SHA256  
・SHA384  
・SHA512  
  
・CRC16-IBM  
・CRC16-CCITT*  
・CRC32  
・CRC64-ECMA-182  
・CRC64-ISO*  
  
・MACTripleDES*  
  
・RIPEMD160*  
  
*がついているものは選択ボックスに直接入力してください。  
  
動作確認

.NET Framework 4.8
