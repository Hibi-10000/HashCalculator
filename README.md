# HashCalculator [v0.5.4]  
  
Hash計算機でHashを計算できます。  
また、ファイルを右クリックしてHash for ContextMenu(F)をクリックし、  
Hashを選択または*をクリックするとそのファイルのHashを計算できます。  
  
対応Hashアルゴリズム  
・MD5  
  
・SHA1  
・SHA256  
・SHA384  
・SHA512  
  
・CRC16-IBM  
・CRC16-CCITT*  
・CRC32  
・CRC64-ECMA  
・CRC64-ISO*  
  
*がついているものは選択ボックスに直接入力してください。  
  
動作確認  
Windows11 Pro 23H2 x86_64  
.NET Desktop Runtime 8.0.3
