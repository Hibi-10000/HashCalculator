# HashCalculator [v0.4.0-alpha]  
  
Hash計算機でHashを計算できます。  
また、ファイルを右クリックしてHash for ContextMenu(F)をクリックし、  
Hashを選択または*をクリックするとそのファイルのHashを計算できます。  
  
対応Hashアルゴリズム  
MD5  
SHA1・256・384・512  
  
動作確認  
Windows10 Pro 21H2 Build19044.1466 64bit  
.NET Framework 4.8
